Validate Your ia.json

Paste your ia.json content below or enter a URL to validate against the official schema.

OR

Try an example:

Command Line Validator

Prefer the terminal? Use the CLI validator.

# Validate a local file
npx ia-json-validator ./ia.json

# Validate a remote URL
npx ia-json-validator https://example.com/ia.json

# JSON output
npx ia-json-validator ./ia.json --format json
